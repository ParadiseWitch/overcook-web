# 食材箱测试清单

## 测试场景：上菜失败后再次拿取食材

### 测试步骤：
1. 从食材箱拿取食材
2. 在切菜板上切好食材
3. 放到盘子上
4. 扔到上菜口（会失败因为只有番茄汤才能上菜成功）
5. 再次从同一个食材箱拿取食材 - **应该能成功拿取**

### 需要测试的食材：

- [ ] 番茄 (B) - 红色圆形
- [ ] 生菜 (L) - 绿色圆形
- [ ] 鱼 (F) - 银色鱼形
- [ ] 米 (R) - 白色方块
- [ ] 洋葱 (O) - 紫色圆形
- [ ] 土豆 (G) - 棕色椭圆
- [ ] 胡萝卜 (P) - 橙色三角
- [ ] 鸡蛋 (Q) - 白色椭圆
- [ ] 紫菜 (N) - 深绿方块（不需要切）

### 预期结果：
所有食材箱在上菜失败后都应该能正常再次拿取食材

### 已实施的修复：
1. ✅ Ingredient.destroy() 现在会从 ALL_ITEMS 中移除
2. ✅ Container.clear() 现在会重新创建 Food 对象
3. ✅ 修复了 Plate.canAddIngredient() 的循环递归问题
