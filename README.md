# 胡闹厨房 web版本 (Overcooked Web)

## 项目简介

一个基于 [Phaser 3](https://phaser.io/) 的web版本《胡闹厨房》(Overcooked) 游戏。目前代码比较凌乱。

链接：<https://paradisewitch.github.io/overcook-web>

## 角色控制

### 玩家 1

* **移动**: `W` `A` `S` `D`
* **互动**: `E`
* **切菜/洗涤**: `R`
* **投掷**: `空格`
* **冲刺**: `Shift`

### 玩家 2

* **移动**: `箭头键`
* **互动**: `O`
* **切菜/洗涤**: `P`
* **投掷**: `L`
* **冲刺**: `Enter`

## 快速开始

### 1. 安装依赖

在项目根目录下运行：

```bash
npm install
```

### 2. 开发模式运行

```bash
npm run dev
```

这将在本地启动一个开发服务器，通常在 `http://localhost:8080` (具体端口可能有所不同) 访问。

### 3. 构建生产版本

```bash
npm run build
```

这将在 `dist` 目录下生成用于部署的优化版本。
